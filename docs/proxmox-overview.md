# Proxmox Overview

This document provides a general overview of the Proxmox-based homelab.

## Platform

The homelab is built on Proxmox VE and uses both LXC containers and virtual machines for service hosting, monitoring, Linux practice and infrastructure experimentation.

## Main Focus Areas

- self-hosted services
- Linux administration
- virtualization
- infrastructure monitoring
- storage organization
- controlled service publishing
- practical documentation

## Workload Types

### LXC Containers
LXC containers are used for lightweight service hosting and operational workloads such as dashboards, monitoring components, backup tools and application services.

### Virtual Machines
Virtual machines are used for Linux practice, Red Hat learning, system testing and general lab experimentation.

## Operational Goals

The environment is designed to help build practical skills in:
- Proxmox administration
- service organization
- monitoring and observability
- documentation
- troubleshooting
- day-to-day infrastructure management

## Notes

This repository focuses on high-level technical documentation and excludes sensitive operational details.
