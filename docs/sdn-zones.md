# SDN Zones

This document provides a high-level overview of the SDN zones used in the Proxmox homelab.

## Current Zones

### localnetwork
Used for the main local lab environment and service connectivity.

### test
Used for testing, separation and experimental configurations.

## Goals

The SDN setup helps:
- organize network segments
- separate workloads when needed
- support testing and experimentation
- improve the overall structure of the lab

## Notes

This is a simplified overview for documentation purposes. Detailed internal networking information is intentionally not published.
